tg_master.tgbuilder¶

This module contains functions to automate creation of BuildFactories and other helpful functions to use in master.cfg

tg_master.tgbuilder.createnamelist(namelist, string)¶

createnamelist creates a list of slaves names which name contains string

Parameters:
  • namelist (list of strings) – List of slave names
  • string (string) – part of slave name to look for
tg_master.tgbuilder.createslavelist()¶

creates a list of slaves from the slaves.cfg file

return:returns list of names of the slave and list of BuildSlave objects
tg_master.tgbuilder.dependencybuilderee(name, pythonpath)¶
dependencybuilderee creates a build factory which downloads the source of the dependency via easy_install and builds the egg.
tg_master.tgbuilder.dependencybuildersvn(svn_url, pythonpath, name, dependencyfromtg=None)¶
dependencybuildersvn creates a build factory which does a checkout from svnURL, downloads the dependencies specified in DependencyFromTG from the TurboGears server, runs setup.py and builds the egg.
tg_master.tgbuilder.doc_builder(**kwargs)¶

Creates documentation of a project. doc_builder takes an package name or an SVN repository as a source.

Parameters:
  • framework (string) – specifies the framework to use: sphinx or epydoc
  • doc_dir (string) – path to the files that should be documentated
  • package (string) – Use this package as source (DocBuilder downloads it from pypi.python.org)
  • svnurl (string) – Use this repository as source
  • needed_conf_file (string) – path to a needed config file that has to be downloaded from the build master
tg_master.tgbuilder.tg10builder(pythonpath, quick=False)¶

tg10builder returns a BuildFactory which downloads TG1.0 from SVN, installs all dependencies, runs the core tests, creates all posible quickstart projects, runs the tests for all of this projects and builds the egg.

Parameters:
  • pythonpath (string) – This is the path, which is used for all python scripts. pythonpath must fit the pattern: “binDir:libDir”.
  • quick (boolean) – Sets quick build mode
