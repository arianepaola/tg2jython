tg_master.ppath¶

This module contains functions to create and modify PYTHONPATHs

tg_master.ppath.splitppath(ppath)¶

Splits the input at the colon and returns the parts

Parameters:
  • ppath (string) – pythonpath that should be splitted
Returns:

parts of the pythonpath
Return type:

list with two elements: bin/script directory and lib directory

Usage

>>> splitppath("bin:lib")
['bin', 'lib']

>>> splitppath("../bin:../lib")
['../bin', '../lib']
tg_master.ppath.modifyppath(ppath, folder)¶

Joins the pythonpath and the folder.

Parameters:
  • ppath (string) – pythonpath that should be extended
  • folder (string) – folder that extends the python path
Returns:

joined pythonpath
Return type:

string

Usage

>>> modifyppath("bin:lib", "..")
'../bin:../lib'
